# Typing Frontline
How to play
-
The goal of this endless game is to gain as many points as you can with your fast typing skills. Your guide will tell you what you should attack with or what should you do while the enemy is approaching. Type the words right as quick as you can to gain more points. If you typo or run out of time, you lose a heart and if you have no hearts left, you lose your run.
After registration you can choose the difficulty, read the instructions, check the leaderboard or start the game.
You need a keyboard to play the game.

JavaScript Techniques
-
- I. Elements of the JavaScript language:
  - let, const variables
  - for loops
  - if, switch conditions
  - arrays, HTML and JavaScript elements
- II. DOM programming:
  - The game field and menus are accessed using document.querySelector() function calls.
  - Points, and scenes are dynamically updated in the DOM (innerText, innerHTML, style).
- III. Event handling details:
  - You can type any letter with the keydown evet. Numbers are not allowed.
  - Buttons work with the basic click event.
- IV. Data storage
  - In-game data stored mostly in variables.
  - Accounts and points are stored in Data Base.
- V. Forms, images, tables:
  - buttons, labels
  - game logo in the menu
  - image backgrounds for certain elements
  - image for the graphics
  - table for the leaderboard
- VI. JavaScript built-in objects
  - Math for round points and for random enemies and words.

PHP Techniques
-
- I. Elements of the PHP language:
  - global and local variables
  - arrays
  - if, switch conditions
  - functions
- II. Server-side data and environment:
  - $_POST data via form
  - fopen, fread
  - connection via PDO
- III. Database operations with PDO:
  - SELECT, INSERT, UPDATE
  - query(), exec()
  - password control with password_hash and password_verify
- IV. Authentication and user logic:
  - check if the username and the password are correct
  - register new user
  - message depends on the result (new user, success, fail)
- V. Response:
  - JSON response
  - dynamic leaderboard
  - date format
- VI. Code organization:
  - every PHP file is unique
